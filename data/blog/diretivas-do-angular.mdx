# Diretivas

Basicamente são extensões da linguagem de **HTML** que permitem a implantação de novos comportamentos de forma declarativa.
Angular contém todos os registros de diretivas atras de procurar se existe alguma para implementar seus comportamentos

## ng-app

-

```html
<!DOCTYPE html>
<html ng-app="litaTelefonica" lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ng-app</title>
    <script src="angular.js"></script>
    <script>
        angular.module("listaTelefone", []);
        angular.module("listaTelefonica").controller("listaTelefonicaCtrl", function ($scope) {
            $scope.app = "Lista Telefonica"
        })
    </script>
</head>
<body ng-controller="listaTelefonicaCtrl">
    <div>
        <h4 ng-bind="app"></h4>
        {{app}}
    </div>
</body>
</html>
```
